# Contributor Code of Conduct

This project adheres to a code of conduct to help create a welcoming and respectful environment for everyone.

## Our Standards

Contributors are expected to:
- Demonstrate empathy and kindness toward other community members.
- Be respectful of differing viewpoints and experiences.
- Provide and accept constructive feedback gracefully.
- Take responsibility for their actions and apologize when necessary.
- Focus on what is best for the community.

The following behavior is unacceptable:
- Harassment or discrimination in any form.
- Personal attacks or insulting comments.
- Publishing private information without permission.
- Disruptive or hostile conduct in project spaces.

## Maintainer Responsibilities

Project maintainers are responsible for:
- Clarifying and enforcing this Code of Conduct.
- Taking appropriate action in response to any violations.
- Removing or correcting contributions that violate policy.
- Temporarily or permanently banning individuals for repeated or severe violations.

## Scope

This Code of Conduct applies within:
- Project repositories
- Issue trackers
- Pull requests and code reviews
- Public or private community channels related to the project

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to project maintainers.  
All complaints will be reviewed fairly and consistently.

## Consequences

Failure to comply with this Code of Conduct may lead to consequences including:
- Warnings
- Temporary or permanent bans
- Reversal of contributions
- Removal from project spaces

## Acknowledgment

This Code of Conduct is adapted from widely used open-source community conduct guidelines promoting a
healthy and respectful ecosystem for contributors.
